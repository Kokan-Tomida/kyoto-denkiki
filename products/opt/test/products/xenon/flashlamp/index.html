ショートアークランプ
Short-arc Lamp

ショートアークランプは、電極間の距離が短いため擬似的な点光源として扱えます。

The short-arc lamp can be handled as a pseudo point source of light because the distance between the electrodes of the lamp is short.

ロングアークランプ
Long-arc Lamp

ロングアークランプは、電極間の距離が長いため発光面のサイズが必要な場合に使用します。

The long-arc lamp is used if a large-sized light emitting surface is required because the distance between the electrodes of the lamp is long.
